1 КРЕДИТ — Serpstat: анализ доменов и страниц конкурентов

Источник: Serpstat API, база g_kz (Google Казахстан) · снимок 2026-07-01. Домены и страницы взяты из конкурентного анализа выдачи. Метрики домена — видимость, органические ключи, оценка трафика/мес, динамика. Метрики страницы — топ-запросы, по которым URL получает трафик.
Ключевые выводы
  • «Залог авто» — слабые конкуренты. Держат нишу автоломбарды (домены с видимостью 2–20, трафик доменов 1–37k/мес; страницы получают всего ~100–360 визитов). Их xT 1.9–3.0. Обгоняется легче всего.
  • Недвижимость / залоговый / рефинанс — стены банков: berekebank, bcc, halyk, altyn, forte ранжируют продуктовые страницы в топ-1..5 с трафиком страницы 250–1200/мес. Домены сильные (видимость 130–940).
  • kmf.kz (МФО) — единственный небанк во всех коммерческих кластерах, тянет трафик статьями (/news-inner). Ближайшая к 1kredit модель.
  • Реструктуризация — все топ-страницы это статьи (home.kz/blog, kmf/news, sberbank, mtsbank). Подтверждает: закрывать блогом.
  • Гиганты вне конкуренции по трафику (kaspi, home.kz, prodengi, 2gis, wikipedia, instagram) — либо агрегаторы/справочники, либо бренды; борьба за топ с ними нецелесообразна.
